# Security Policy

## Supported Versions
This is a fun project provided as-is, with no guarantees or warranties. Security updates may be provided at the discretion of the maintainers.

## Reporting a Vulnerability
If you discover a security vulnerability, please report it by emailing the maintainers at procionegobbo@gmail.com.

- Do not disclose security issues publicly until they have been reviewed and patched (if applicable).
- Provide as much detail as possible to help us reproduce and address the issue.

## Response
We will address security reports as soon as possible, but no specific response timeframe is guaranteed.
